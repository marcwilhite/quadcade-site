# quadcade-site

Marketing, privacy, and support pages for the iOS game [Quadcade](https://github.com/marcwilhite/quadcade).

Published via GitHub Pages:

- Marketing landing: **https://marcwilhite.github.io/quadcade-site/**
- Privacy policy: **https://marcwilhite.github.io/quadcade-site/privacy/**
- Support & FAQ: **https://marcwilhite.github.io/quadcade-site/support/**

The privacy URL is the one referenced in App Store Connect under the app's Privacy Policy field. The support URL goes in the Support URL field.

## Editing

Pages live as `*.md` at the repo root. Edit, commit, push to `main`, and GitHub Pages republishes within ~1 minute.

Screenshots in `screenshots/` are the 1284×2778 App Store assets — shared between this site and App Store Connect.
